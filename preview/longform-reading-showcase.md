# Longform Reading Showcase

The longform reading surface is for essays, chapters, travel notes, serialized fiction drafts, and reflective writing. It should not behave like the research mode: there are no citation chips or evidence tables here. The priority is paragraph rhythm, chapter spacing, gentle scene breaks, and a page that invites sustained reading.

> A long page should feel like a quiet room: enough structure to keep orientation, not enough ornament to interrupt the sentence.

## Chapter One — Warm Paper

This opening paragraph tests the lead-paragraph treatment. It should have slightly more presence than normal body copy, and the first letter should give the chapter a quiet editorial entry point without turning the page into a magazine cover.

Longform prose needs room for breath. Paragraphs should not stack like compact notes, but they also should not become loose blog cards. The mode should preserve Ouroboros' readable line width and warm contrast while making the page feel slower, steadier, and more intentional.

Dialogue should remain comfortable inside the same rhythm:

> “Keep the edges soft,” the note says, “but let the structure do the work.”

A list may appear inside a draft, but it should stay secondary to prose:

- chapter sections need visible distance
- scene breaks should feel deliberate
- images should sit calmly inside the page
- callouts should not dominate the reading surface

## Chapter Two — The Return

A second chapter heading should again create a clear pause. The first paragraph after a chapter heading receives the same lead treatment, while subsequent paragraphs return to a calmer baseline.

Images, when present, should feel like plates inside a book rather than raw attachments. Tables should usually be avoided in this mode; when a draft needs evidence tables, use Research reading mode instead.
