# Tasks Showcase

This note checks standard and extended task states plus progress bars.

## Standard Tasks

- [ ] Open task
- [x] Completed task
- [/] In progress
- [-] Cancelled / dropped
- [!] Important
- [?] Question
- [*] Starred
- ["] Quote

## Extended States

- [<] Scheduled
- [>] Forwarded
- [I] Idea
- [b] Bookmark
- [c] Con
- [d] Down
- [f] Fire
- [i] Information
- [k] Key
- [l] Location
- [p] Pro
- [u] Up
- [w] Win
- [P] Open PR
- [M] Merged PR
- [D] Draft PR
